Landing Page Api
=========

An api to capture signups on a landing page

Using ExpressJS, Mongoose, Chai and Babel and deployed on Heroku

## Installation

```shell
  brew install mongodb
  npm install
  npm start
```

## Tests

```shell
  npm run test
```

## Deployment

```shell
  heroku git:remote -a landing-page-api
  git push heroku master
```
